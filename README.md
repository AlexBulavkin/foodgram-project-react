### Foodgram
Проект Foodgram позволяет опубликовать свой любимый рецепт, посмотреть рецепты других пользователей и получить список покупок в один клик.

### Стек технологий
Python, gunicorn, Nginx, Docker, docker-compose, PostgreSQL
